# RN Learning Journey

My 5-month roadmap from web fundamentals to a production-ready mobile app.

## Goal

1 app published on Google Play Store by August 2026.

## Stack

`JavaScript / TypeScript` · `React` · `React Native + Expo` · `Firebase` · `Zustand`

## Structure

### Month 1 — Web Foundations ✅
- JavaScript & TypeScript fundamentals.
- HTML structure and React basics.

### Month 2 — Styling & React 
- CSS3 and Flexbox.
- Advanced React: hooks, props, component architecture.

### Month 3 — React Native & Expo
- Transition from web to native.
- Mobile UI/UX and navigation.

### Month 4 — Data & State
- Firebase authentication and Firestore.
- Global state with Zustand.

### Month 5 — Budget Tracker App 📱
- Full build, UI polish, performance optimization.
- Play Store launch.
